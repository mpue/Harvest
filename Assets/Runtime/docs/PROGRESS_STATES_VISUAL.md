## Accessibility Considerations

### Color Blind Modes
- **Protanopia/Deuteranopia**: Use Blue theme
- **Tritanopia**: Use Yellow ? Red theme
- **Monochrome**: Add pattern overlay (stripes)

### Visual Indicators Beyond Color
? **Progress Text**: Shows exact percentage
? **Rotating Indicator**: Movement-based feedback
? **Dimming Effect**: Brightness change
? **Glow Pulse**: Intensity variation
